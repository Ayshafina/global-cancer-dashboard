# Global Cancer Patient Analytics Dashboard

## 📊 Project Overview
This project analyzes global cancer patient data using Power BI and Excel.

## 🛠 Tools Used
- Power BI
- Excel

## 📁 Dataset
Dataset.xlsx contains patient details like cancer type, causes, and demographics.

## 📈 Dashboard Features
- Patient Details Analysis
- Causes of Cancer Visualization
- Global Trends

## 🎯 Key Insights
- Identified major cancer causes
- Visualized patient distribution
- Created interactive dashboard
